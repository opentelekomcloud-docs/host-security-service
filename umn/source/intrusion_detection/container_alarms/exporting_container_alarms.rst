Exporting Container Alarms
==========================

You can export container alarms and events to a local PC.


Exporting Container Alarms
--------------------------

#. Log in to the management console.
#. Click |image1| in the upper left corner of the page, select a region, and choose **Security** > **Host Security Service**. The HSS page is displayed.
#. In the navigation pane, choose **Intrusion Detection** > **Alarms**.

4. Click the **Container Alarms** tab.
5. Click **Export** above the alarm list to export all security events.

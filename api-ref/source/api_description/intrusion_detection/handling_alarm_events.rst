Handling Alarm Events
=====================

Function
--------

This API is used to handle alarm events.

URI
---

POST /v5/{project_id}/event/operate

.. table:: **Table 1** Path Parameters

   +-----------------+-----------------+-----------------+-----------------+

   |                 |                 |                 |                 |
   |                 |                 |                 | Minimum: **20** |
   |                 |                 |                 |                 |
   |                 |                 |                 | Maximum: **64** |
   +-----------------+-----------------+-----------------+-----------------+
